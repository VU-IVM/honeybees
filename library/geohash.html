plot_geohash_shifts(lon=4.8945, lat=52.3667, radius=5000, bits=31, show=True)[source]ÔÉÅ

This function can be used to explore how geohashes can be used to cover a given circle. There is a trade-off between precision of the geohash, speed and how well the shifts represent a circle. This function plots a circle and geohash windows required to cover the circle.

Parameters:

  • lon (float) ‚Äì Longitude of circle centre.

  • lat (float) ‚Äì Latitude of cirle centre.

  • radius (Union[float, int]) ‚Äì Radius of circle.

  • bits (int) ‚Äì Number of geohash bits used.

  • show (bool) ‚Äì Whether to show the resulting plot.

Return type:

None
